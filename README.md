# CyberChef Lite (Browser Script)

一个轻量级的浏览器划词解码工具。无需安装额外的浏览器扩展，只需 Violentmonkey/Tampermonkey 脚本管理器即可运行。

## ✨ 主要功能

*   **零侵入交互**：选中网页文本后出现悬浮图标（⚡️），点击后才加载界面，不影响正常浏览。
*   **多重解码器**：集成 Hex、URL、Base64 等常见编码转换。
*   **PowerShell 专精**：内置 **PowerShell Base64 (UTF-16LE)** 解码，解决常规解码器乱码问题。
*   **高性能**：针对大段 Hex 流（MB级别）优化，使用 `TypedArray` 处理，浏览器不卡顿。
*   **智能交互**：
    *   支持**拖拽移动**面板，避免遮挡内容。
    *   支持**原地替换**，将解码后的明文直接替换网页上的乱码。
    *   支持**递归解码**（套娃模式），结果可作为下一次解码的输入。
*   **样式隔离**：使用 Shadow DOM 技术，不破坏原网页的 CSS 布局。

## 🛠 支持的编码格式

| 按钮名称 | 功能描述 | 典型场景 |
| :--- | :--- | :--- |
| **Hex->Str** | 十六进制转字符串 (UTF-8) | 流量包 payload、二进制数据 |
| **URL Dec** | URL 解码 | GET 请求参数、Web 日志 |
| **Base64** | 标准 Base64 解码 (UTF-8) | Web 通用编码 |
| **PS Base64** | **PowerShell Base64 (UTF-16LE)** | 恶意脚本分析、Powershell `-Enc` 参数 |
| **Unicode** | Unicode 转义解码 (`\u4e2d`) | JSON 数据、Java 异常堆栈 |
| **Smart** | 智能尝试 | 自动探测并尝试解码 |

## 🚀 安装方法

1.  安装浏览器扩展 **Violentmonkey** (暴力猴) 或 **Tampermonkey** (油猴)。
2.  点击扩展图标 -> **添加新脚本**。
3.  将 JS 代码完整粘贴到编辑器中。
4.  按 `Ctrl + S` 保存即可。

## 📖 使用指南

1.  **触发**：在网页上用鼠标划选一段乱码（如 `48656c6c6f`）。
2.  **打开**：点击鼠标选区旁出现的蓝色闪电图标 **⚡️**。
3.  **解码**：
    *   点击面板上的对应按钮（如 `Hex->Str`）。
    *   结果会显示在黑色文本框中。
4.  **进阶操作**：
    *   **移动面板**：按住面板顶部的灰色标题栏即可拖拽。
    *   **套娃解码**：如果解出的结果里还包含 URL 编码，直接点击 `URL Dec` 即可对当前结果继续解码。
    *   **替换原文**：点击橙色的 `替换原文` 按钮，网页上的乱码会直接变成明文（方便阅读长篇日志）。

## ⚠️ 注意事项

*   **替换原文功能**：仅在 DOM 结构允许的情况下生效（如 input 框或普通文本节点）。
*   **隐私安全**：所有解码逻辑均在本地浏览器 JS 引擎中运行，**无任何网络请求**，数据绝对安全。
